# 费马小定理

假如 a 是一个整数， p 是一个质数，那么 $a^ {p}-a$ 是 p 的倍数，也可以表示为 $a^ {p} \equiv a (\mod p)$ 。

如果 a 不是 p 的倍数的话，则 $a^ {p-1} \equiv 1 (\mod p)$ 。

# 威尔逊定理

$(p-1)! \equiv -1\pmod  p$ 是 p 为素数 的 充分必要条件。
