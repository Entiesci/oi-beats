有向图游戏
公平组合游戏都可以转换为有向图游戏。


mex运算
$mex(S)=min\{x| x \in N,x \notin S\}$


SG函数
$SG(x)=mex(\{SG(y_1),SG(y_2),...,SG(y_k)\}) $


SG定理
当$SG(s_1)\oplus SG(s_2)\oplus... \oplus SG(s_n)\ne 0 $时，先手必胜；反之，先手必败。
